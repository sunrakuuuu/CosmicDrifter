# **App Name**: Cosmic Drifter

## Core Features:

- Story Mode Campaign: Progress through levels with unique alien types, obstacles, and boss fights to find a cure for the zombie outbreak on Earth, incorporating short dialogues or cutscenes between levels.
- Endless Mode: Survive infinitely against randomly spawning alien ships and obstacles with increasing difficulty, earning a score for each alien destroyed.
- Adaptive Difficulty Scaling: The AI tool dynamically adjusts enemy spawn rates and attack patterns based on the player's skill level to maintain engagement.
- Intuitive Touch Controls: Control the spaceship by dragging, with optional tap-to-shoot functionality for a seamless mobile experience.
- Dynamic Visual Feedback: Respond to the players' actions by using movement and spawning behaviors that reward skilled gameplay, making sure not to overwhelm the player with enemies.
- Power-Up System: Collect shields, rapid fire, and health recovery power-ups to enhance the spaceship's capabilities.
- Animated Starfield Background: Stars moving downward simulate speed to give a stronger sense of motion.

## Style Guidelines:

- Primary color: Vibrant Purple (#A050BE) for a cosmic and energetic feel, suggesting both the unknown and a sense of discovery in space.
- Background color: Deep Charcoal (#2A2A2E), very desaturated from the primary, allowing the gameplay elements to pop while still meshing with the darkness of space
- Accent color: Electric Blue (#50B0BE) to draw attention to key elements such as power-ups, the current score, and user interface prompts.
- Headline font: 'Space Grotesk' sans-serif; Body font: 'Inter' sans-serif; Pair for a computerized and neutral look that provides the game with both great legibility as well as distinct science-fiction flavor.
- Simple, glowing icons representing power-ups and UI elements.
- Score display at the top with controls (if any) at the bottom for a mobile-friendly interface.
- Smooth animations for spaceship movement, alien spawns, and power-up effects.
